# japanese-holidays-api
内閣府公式の syukujitsu.csv (祝日データ) をUTF-8とJSONに変換 | UTF-8/JSON API for Japanese national holidays from Cabinet Office's official CSV "syukujitsu.csv".

## データソース / Data Source

このプロジェクトは内閣府が公開している祝日データ（syukujitsu.csv）を利用しています。

This project uses the official Japanese national holiday data (syukujitsu.csv) published by the Cabinet Office of Japan.

- **出典 / Source**: 内閣府「国民の祝日について」/ Cabinet Office "About National Holidays"
  - URL: https://www8.cao.go.jp/chosei/shukujitsu/gaiyou.html
