Japanese Holidays API

日本の祝日データを提供するAPIです。内閣府の公式データ（Shift_JIS）をUTF-8とJSONに変換して配信しています。

API for Japanese national holidays. Converts official Cabinet Office data from Shift_JIS to UTF-8 and JSON.

API Endpoints

  • holidays.json - JSON形式の祝日データ / Holiday data in JSON format
  • holidays.csv - UTF-8 CSV形式の祝日データ / Holiday data in UTF-8 CSV format

使用例 / Usage Example

JavaScript/Fetch API

fetch('https://nysg.github.io/japanese-holidays-api/holidays.json')
  .then(response => response.json())
  .then(data => {
    console.log(data);
    // [{ "date": "1955/1/1", "name": "元日" }, ...]
  });

curl

# JSON
curl https://nysg.github.io/japanese-holidays-api/holidays.json

# CSV
curl https://nysg.github.io/japanese-holidays-api/holidays.csv

データソース / Data Source

このプロジェクトは内閣府が公開している祝日データ（syukujitsu.csv）を利用しています。

This project uses the official Japanese national holiday data (syukujitsu.csv) published by the Cabinet Office of Japan.

更新頻度 / Update Frequency

毎週日曜日0時（UTC）に自動更新されます。

Data is automatically updated every Sunday at 0:00 UTC.
